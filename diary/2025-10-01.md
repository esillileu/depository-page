# 주요 작업 및 결과
- [[NVIDIA AI ACADEMY]] 파이썬 수업
	-  [[With Statement|with 문]]
		- [[Context Manager|컨텍스트 매니저]] 객체를 지정하여 사용함
			- `__enter__()`, `__exit__()` 메서드가 정의된 객체
			- 미리 선언된 컨텍스트 매니저도 사용 가능
			- `as`를 이용한 [[Aliasing|얼라이어싱]]
			- [[With Statement|with 문]]문 종료시 지정된 객체에 대해 자동으로 `__exit__()` 사용
		- [[Scope|스코프]] 생성 안함
			- 내부에서 선언된 [[Identifier|식별자]]를 밖에서 사용 가능
			- 종료한 컨텍스트 매니저도 사라지지 않음
	- `yield`
		- 포함된 함수를 [[Generator Function|제너레이터 함수]]로 정의함
			- 결과와 상관 없이 [[Generator Object|제너레이터 객체]] 반환 
				- 생성된 제너레이터가 `return` 만나면 `StopIteration` 예외에 값을 넣어 `raise`
					- `return`이 포함된 제너레이터 함수로 만든 제너레이터를 `yield from`으로 위임하면 `return` 에 있는 값도 받을 수 있음
		- [[Expression|표현식]]을 만듬
			- 표현식 - 값을 만들어내는 코드 조각
			- 문 - 동작하는 코드 조각
		- `yield <value>`
			- 호출자에 값을 반환
			- 일시 중단
			- `<generater>.send(<value>)`로 재 호출 시 호출자의 전달 값으로 표현식 치환 
		- `yield from <iterater>`
			- 표현식 평가 시 내부 이터레이터에 대한 평가로 이동
			- 이터레이터에 대한 이터레이션이 끝날 때까지 내부 이터레이터 평가
			- 내부 이터레이터의 결과로 표현식 치환
