# Span
> [[Vector Space|벡터공간]]의 부분 집합을 통해 [[Subspace|부분공간]] 생성
## 정의
+ ***생성***은 [[Vector Space|벡터공간]] $V$의 부분 집합 $S$의 원소들의 [[Linear Combination|일차 결합]]으로 $V$에 대한 [[Subspace|부분공간]]을 만드는 연산
	+ $$W = {span}(S) = <S>$$
## 성질
+ [[Vector Space|벡터공간]]은 생성 가능
