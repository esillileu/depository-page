# Homogeneous System of Linear Equation
> 상수항이 0인 [[System of Linear Equation|선형연립방정식]]
## 정의
+ ***동차선형연립방정식***은 상수항이 0인 [[System of Linear Equation|선형연립방정식]]
	+ $$\begin{matrix}
a_{11}x_{1}+a_{12}x_{2}+\cdots+a_{1n}x_{n} = 0\\
a_{21}x_{1}+a_{22}x_{2}+\cdots+a_{2n}x_{n} = 0\\
\vdots \\
a_{m1}x_{1}+a_{m2}x_{2}+\cdots+a_{mn}x_{n} = 0
\end{matrix}$$
## 성질
+ 자명한 해와 자명하지 않은 해를 가짐
	+ 자명한 해
		+ $$x_i=0$$
	+ 자명하지 않은 해 - 무수히 많은 해
		+ 미지수가 방정식보다 많으면 자명하지 않은 해를 가짐
+ 동차선형연립방정식 $A\mathbf x = \mathbf 0$의 [[Solution Space|해공간]] $W$의 [[Vector|벡터]]는 $A$의 행[[Vector|벡터]]에 [[Orthogonal Vectors|직교]]
