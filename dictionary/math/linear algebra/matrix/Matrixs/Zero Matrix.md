# Zero Matrix
> 영행렬
## 정의
+ ***영행렬***은 모든 [[Component|성분]]이 0인 임의 크기의 행렬 $O$
	+ $$O = [0]_{m\times n}$$
## 성질
+ 임의의 [[Matrix|행렬]] $A$와 크기가 같은 영행렬 $O$에 대하여 다음이 항상 성립
	+ [[Sum of Matrix|행렬합]]의 [[Binary Operation#항등원|항등원]]
		+ $$A+O = O+A = A$$
	+ [[Product of Matrix|행렬곱]]의 [[Binary Operation#영원|영원]]
		+ $$A\times O = O\times A = O$$
