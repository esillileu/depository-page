# Matrix Sum
> 행렬의 합
## 정의
+ ***행렬합***은 크기가 같은 두 행렬 $A$, $B$를 [[Element-Wise|성분끼리]] 더하는 행렬의 [[Binary Operation|이항연산]]
	+ $$\begin{matrix}
A = [a_{ij}]_{m\times n} \\
B = [b_{ij}]_{m\times n} \\
\end{matrix}$$
	+ $$A+B = [a_{ij}+b_{ij}]_{m\times n}$$
