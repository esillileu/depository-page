# Row Echelon Form
> [[System of Linear Equation|선형연립방정식]]의 해를 구하기 쉬운 [[Coefficient Matrix|계수행렬]]의 형태
## 정의 
+ ***REF***는 다음 조건을 만족하는 $m\times n$ [[Matrix|행렬]] $E$ 
	+ [[Leading Entry|선행성분]]이 없는 열은 아래쪽에 배치
	+ [[Leading Entry|선행성분]]은 모두 1
	+ [[Leading Entry|선행성분]]은 아래 행의 [[Leading Entry|선행성분]]보다 왼쪽에 존재
