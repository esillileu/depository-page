# # Scalar Multiplication of Matrix
> 행렬의 스칼라배
## 정의
+ ***행렬의 스칼라배***는 크기가 행렬 $A$의 각 [[Component|성분]]에 대해 실수 $k$를 곱하는 [[Function|함수]]
	+ $$\begin{matrix}
A = [a_{ij}]_{m\times n} \\
\end{matrix}$$
	+ $$kA = [k\times a_{ij}]_{m\times n}$$
