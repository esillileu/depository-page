# Matrix Equality
> 동일한 행렬
## 정의
+ ***동일한 행렬***은 두 행렬 $A$, $B$에 대해 다음을 만족하는 $A$, $B$의 관계
	+ $$\begin{matrix}
A = [a_{ij}]_{m\times n} \\
B = [b_{ij}]_{m\times n} \\
\end{matrix}$$
	+ 	$$\begin{matrix}
A = B \\
\forall i \leq m,\;\;\forall j \leq n, \;\;a_{ij} = b_{ij} 
\end{matrix}$$
