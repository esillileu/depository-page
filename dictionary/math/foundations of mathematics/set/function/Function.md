# Function
> 두 [[Set|집합]]의 관계
## 정의 
+ ***[[Set|집합]] $X$, $Y$에 대한 함수 $f$*** 는 다음 조건을 만족하는 $X$, $Y$의 원소로 구성된 순서 쌍의 [[Set|집합]]  
	+ $$\forall (x, y) \in f \wedge \forall (x, y') \in f \Rightarrow y = y'$$
	+ $$f=\{(x, y)|\;x \in X,\;y \in Y\}$$
	+ $X$는 [[Domain|정의역]]
	+ $Y$는 공역
	+ $f(X)$는 치역
