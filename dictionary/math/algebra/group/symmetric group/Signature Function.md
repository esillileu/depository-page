# Signature Function
> [[Permutation|치환]]을 1과 -1에 대응시키는 함수
## 정의 
+ ***부호화함수***는 [[Permutation|치환]] $\sigma$가 [[Even Permutation|짝치환]] 1, [[Odd Permutation|홀치환]]일때 -1에 대응시키는 [[Function|함수]] 
	+ $$\operatorname{sgn}(\sigma) = (-1)^{\operatorname{inv}(\sigma)}$$
