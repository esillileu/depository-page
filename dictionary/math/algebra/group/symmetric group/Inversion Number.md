# Inversion Number
> [[Permutation|치환]]에서 [[Inversion|반전]]이 일어난 횟수
## 정의 
+ ***반전수***는 [[Permutation|치환]] $\sigma$에서 [[Inversion|반전]]이 일어난 횟수 
	+ $$
\operatorname{inv}(\sigma) = \left| \{ (i, j) | 1 \leq i < j \leq n, \sigma_i > \sigma_j \} \right|

$$
