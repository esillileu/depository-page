# Inverse Element
> [[Binary Operation|이항연산]]에서 연산 결과로 [[Identity Element|항등원]]을 만들어내는 원소
### 역원
+ ***[[Set|집합]] $S$ 위의 이항연산 $\ast$ 의 역원***은 $S$와 그 위의 이항연산 $\ast$ 의 항등원 $e$에 대해 다음을 만족하는 $a'$
	+ $$\begin{matrix}
a * a' = a' * a = e \\
a, b\in S
\end{matrix}$$
