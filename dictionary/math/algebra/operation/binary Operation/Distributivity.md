# Distributivity
> 두 개의 [[Binary Operation|이항연산]]에 대해 분배되는 성질
### 분배법칙
+ ***[[Set|집합]] S위의 이항연산 $+$에 대해 분배법칙을 만족하는 $S$위의 이항연산 $\ast$*** 은 다음을 항상 만족하는 것 
	+ $a, b, c \in S$
	+ 좌분배법칙 만족$$a * (b + c) = (a * b) + (a * c)$$
	+ 우분배법칙 만족$$(b + c) * a = (b * a) + (c * a)$$
