# Binary Operation
> 특정 [[Set|집합]]의 두 개의 [[Operand|피연산자]]로부터 결과를 얻어내는 연산
## 정의 
+ ***[[Set|집합]] $S$ 위의 이항연산***은 $S$의 임의의 두 원소로 이루어진 순서쌍을 $S$의 원소에 하나씩 대응시키는 [[Function|함수]]
	+ $$\begin{matrix}
* : S \times S \to S \\
(a, b) \mapsto a*b \\
a, b\in S
	\end{matrix}$$
