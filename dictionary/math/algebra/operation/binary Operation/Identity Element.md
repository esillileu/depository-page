# Identity Element
> [[Binary Operation|이항연산]]에서 [[Set|집합]] 내 모든 원소와의 연산 결과를 재귀시키는 원소
### 항등원
+ ***[[Set|집합]] $S$ 위의 이항연산 $\ast$ 의 항등원***은 $S$에 대해 다음을 항상 만족하는 $e$
	+ $$\begin{matrix}
a * e = e * a = a\\
a, b\in S
\end{matrix} $$
