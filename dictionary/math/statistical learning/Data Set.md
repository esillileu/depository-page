# Data Set
> 데이터셋
## 정의 
+ ***데이터셋***은 [[Ground Truth|실제 분포]]에서 [[Sampling|샘플링]] 한  [[Input Space|입력 공간]] $\mathcal X$의 원소 $x$와 [[Target Space|대상 공간]] $\mathcal Y$의 원소$y$ 쌍의 [[Finite Set|유한 집합]]
	+ $$\begin{matrix}
\mathcal D(p) \sim p\\
p = P_{X, Y}, X \in \mathcal X, Y \in \mathcal Y
\end{matrix}$$
